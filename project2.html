Parking System with GUI and DB

Project Overview

The Interactive Parking System is a Java application designed to manage and facilitate parking in a parking lot. This application provides an interactive user interface for parking lot operators and allows them to keep track of parked vehicles, their entry and exit times, and the number of available parking slots. The system is backed by a MySQL database to store information about parked vehicles.

Key Features

  • Interactive user interface for parking management
  • Database integration using MySQL to store vehicle information
  • Tracking of entry and exit times for vehicles
  • Real-time availability of parking slots

Technologies Used

  • Java for the application logic
  • MySQL for database management
  • Swing for the graphical user interface
  • JDBC for database connectivity
