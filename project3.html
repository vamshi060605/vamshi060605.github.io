Raspberry Pi Router Configuration

Project Overview

Configured a Raspberry Pi to act as a home router using OpenWrt, optimizing it for secure and reliable network connectivity. This project involved setting up static IP addresses, DNS servers, and custom routing rules, along with remote management via SSH.

Key Objectives

  • Set up a Raspberry Pi as a router using OpenWrt.
  • Configure static IP addresses and DNS for efficient network management.
  • Implement security measures for remote access via SSH.

Technologies Used

  • OpenWrt
  • Raspberry Pi
  • Linux, SSH, TCP/IP, DNS, DHCP
