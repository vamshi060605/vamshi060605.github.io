Netflix Clone Project

Project Overview

This project is a Netflix Clone developed using HTML, CSS, JavaScript, and React. It replicates the UI of Netflix and allows users to browse a selection of movie posters, view details, and play trailers.

Key Features

  • Interactive user interface similar to Netflix
  • Responsive design across devices (mobile, tablet, desktop)
  • Dynamic movie details page with trailers
  • Used React components for modular design

Technologies Used

  • React JS
  • HTML, CSS, JavaScript
  • Bootstrap for responsive design
